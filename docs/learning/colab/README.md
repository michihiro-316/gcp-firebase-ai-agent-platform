# Google Colab ノートブック

このフォルダには、Google Colabで実行できる学習用ノートブックがあります。

---

## ノートブック一覧

| ファイル | 内容 |
|----------|------|
| `01_Python基礎とバックエンド理解.ipynb` | Python基礎（辞書、例外処理、async/await等） |
| `02_TypeScriptとReact基礎.ipynb` | TypeScript/ReactをPythonと比較して学ぶ |

---

## 使い方

### 方法1: GitHubから直接開く

1. GitHubでノートブックを開く
2. 「Open in Colab」バッジをクリック

### 方法2: Google Colabにアップロード

1. [Google Colab](https://colab.research.google.com/) を開く
2. 「ファイル」→「ノートブックをアップロード」
3. `.ipynb` ファイルを選択

### 方法3: Google Driveから開く

1. `.ipynb` ファイルをGoogle Driveにアップロード
2. ファイルをダブルクリック
3. 「Google Colaboratory で開く」を選択

---

## Google Colabとは？

- Googleが提供する無料のPython実行環境
- ブラウザだけでPythonコードを実行できる
- インストール不要ですぐ始められる

---

## ノートブックの進め方

1. 上から順番にセルを実行（Shift + Enter）
2. コードを変更して実験してみる
3. わからないところはコメントを読む
