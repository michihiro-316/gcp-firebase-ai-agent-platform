# 学習ガイド

このフォルダは、初心者エンジニアがシステムを**1〜2日で理解する**ための勉強資料です。

---

## はじめに

**まず読むファイル**: `md/00_はじめに読んでください.md`

---

## 学習アプローチ

このガイドは「フロー別学習」を採用しています。

```
【従来】ファイル単位で学ぶ
  main.py → auth.py → agent.py
  ❌ ファイル間の繋がりがわかりにくい

【新方式】フロー単位で学ぶ
  「チャットを送信したら何が起きるか」を追う
  ✅ コードの繋がりがわかる！
```

|----------|------|

1. [Google Colab](https://colab.research.google.com/) を開く
2. 「ファイル」→「ノートブックをアップロード」
3. `.ipynb` ファイルを選択
4. セルを上から順に実行（Shift + Enter）
