# 設計資料

このフォルダは、AIチャットシステムの**設計・実装ガイド**です。

---

## はじめに

**まず読むファイル**: `md/01_はじめに読んでください.md`

---

## 資料の目的

この設計資料は以下の用途を想定しています：

- **技術提案**: エンジニアへの導入提案資料として
- **開発ガイド**: 本システムをベースにした開発の参考として
- **オンボーディング**: 新規メンバーへの技術説明資料として

---

## フォルダ構成

```
learning/
├── md/                 ← Markdownドキュメント
│   ├── 01_はじめに読んでください.md    ← まずこれ！
│   ├── 02_全体像.md                   ← アーキテクチャ
│   ├── 03_バックエンド解説.md          ← Python実装
│   ├── 04_フロントエンド解説.md        ← React実装
│   ├── 05_セットアップの流れ.md        ← 環境構築
│   ├── FLOW_01_チャット送信の流れ.md   ← 詳細フロー
│   └── ...
├── colab/              ← Google Colabノートブック
└── github-pages/       ← HTML化されたドキュメント
```

## 補足資料

### Google Colab（`colab/`）

| ファイル | 内容 |
|----------|------|
| `01_Python基礎とバックエンド理解.ipynb` | async/await、デコレータなど |
| `02_TypeScriptとReact基礎.ipynb` | Pythonとの比較で学ぶ |

使い方:
1. [Google Colab](https://colab.research.google.com/) を開く
2. 「ファイル」→「ノートブックをアップロード」
3. `.ipynb` ファイルを選択
4. セルを上から順に実行（Shift + Enter）
